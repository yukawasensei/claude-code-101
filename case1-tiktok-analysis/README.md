# Case 1: TikTok 视频数据分析

## 案例背景

你是一名 TikTok 短视频运营人员，负责一个电商品牌的 TikTok 账号。过去一个月发布了 50 条视频，现在需要分析这些视频的表现，找出高转化视频的特征，为下个月的内容策略提供数据支持。

## 学习目标

通过本案例，你将学会：
- 使用 Claude Code 进行数据探索和清洗
- 进行多维度的统计分析
- 创建交互式数据可视化图表
- 生成专业的数据分析报告

## 数据说明

### 数据文件
- **文件名**: `tiktok_data.csv`
- **记录数**: 50 条
- **时间范围**: 2026年1月1日 - 2026年1月31日

### 数据字段

| 字段名 | 说明 | 示例 |
|--------|------|------|
| video_id | 视频ID | V001 |
| title | 视频标题 | 3个护肤小技巧，让你的皮肤水嫩透亮 |
| publish_date | 发布日期 | 2026-01-01 |
| publish_hour | 发布小时 | 18 |
| views | 播放量 | 15000 |
| likes | 点赞数 | 1200 |
| comments | 评论数 | 85 |
| shares | 分享数 | 120 |
| duration | 视频时长（秒） | 45 |
| video_type | 视频类型 | 教程 |
| conversions | 转化量 | 25 |

### 视频类型说明
- **教程**: 教学类内容（如"3个护肤技巧"）
- **开箱**: 产品开箱展示
- **测评**: 产品对比测评
- **剧情**: 情景剧形式
- **直播切片**: 直播精彩片段

## 使用步骤

### 1. 准备数据

如果还没有数据文件，可以使用生成脚本创建：

```bash
python generate_data.py
```

这将生成 `tiktok_data.csv` 文件。

### 2. 启动 Claude Code

```bash
cd case1-tiktok-analysis
claude
```

### 3. 数据探索

向 Claude 提问：

```
请帮我分析 tiktok_data.csv 文件：
1. 数据概览（行数、列数、字段类型）
2. 检查是否有缺失值
3. 显示前5行数据
```

### 4. 统计分析

继续提问：

```
请进行以下统计分析：
1. 各视频类型的数量分布
2. 播放量、点赞数、转化量的基本统计（均值、中位数、最大最小值）
3. 不同视频类型的平均转化量对比
4. 发布时间段与转化量的关系
```

### 5. 可视化分析

请求生成图表：

```
请生成以下可视化图表：
1. 各视频类型的转化量对比（柱状图）
2. 播放量与转化量的关系（散点图）
3. 不同发布时间段的转化效果（折线图）
4. 视频时长分布（直方图）
```

### 6. 深度分析

进行相关性分析：

```
请分析：
1. 播放量、点赞数、评论数、分享数与转化量的相关性
2. 找出转化率最高的前10个视频，分析它们的共同特征
3. 识别表现异常的视频（转化特别高或特别低）
```

### 7. 生成报告

最后生成分析报告：

```
请基于以上分析，生成一份TikTok视频运营分析报告，包括：
1. 数据概览
2. 关键发现
3. 高转化视频的特征
4. 下月内容策略建议
```

## 常见问题

### Q: 数据文件在哪里？
A: 运行 `python generate_data.py` 生成，或者使用已有的 `tiktok_data.csv`。

### Q: 图表无法显示怎么办？
A: 确保安装了必要的 Python 库（pandas, matplotlib, seaborn）。Claude Code 会自动安装缺失的库。

### Q: 如何修改分析维度？
A: 直接告诉 Claude 你想分析的维度，例如"请分析视频时长与转化量的关系"。

### Q: 可以用自己的数据吗？
A: 可以！只要数据格式类似（CSV 格式，包含相关字段），就可以使用相同的分析方法。
