# 直播间复盘分析框架

## 核心指标体系

### 流量层
| 指标 | 计算方式 | 健康基准（参考） | 预警阈值 |
|---|---|---|---|
| 场次 UV | 平台直接提供 | — | 环比下滑 >20% |
| 在线峰值 | 平台直接提供 | — | 环比下滑 >30% |
| 平均在线 | 总在线人次 / 时长 | — | — |
| 流量来源占比 | 自然流 / 付费流 / 粉丝流 | 自然流 >40% 为佳 | 付费流 >80% 需警惕 |
| 新粉占比 | 新进粉丝 / 总 UV | >30% 表明拉新能力强 | <10% 说明内容圈层固化 |

### 留存层
| 指标 | 计算方式 | 健康基准 | 预警阈值 |
|---|---|---|---|
| 人均停留时长 | 总停留时长 / UV | >3 分钟为佳 | <1 分钟需优化开场 |
| 停留时长分布 | 0-30s / 30s-3min / 3min+ 占比 | 3min+ 占比 >20% | 0-30s 占比 >60% |

### 互动层
| 指标 | 计算方式 | 健康基准 | 预警阈值 |
|---|---|---|---|
| 互动率 | (弹幕+点赞+分享) / UV | >15% | <5% |
| 弹幕率 | 弹幕人数 / UV | >8% | <3% |
| 关注转化率 | 新增关注 / UV | >3% | <1% |

### 转化层
| 指标 | 计算方式 | 健康基准 | 预警阈值 |
|---|---|---|---|
| 下单转化率 | 下单人数 / UV | 服装类 >3%，食品类 >5% | <1% 需检查选品/话术 |
| 客单价 | GMV / 订单数 | 依品类而定 | 客单价下滑 >15% |
| UV 价值 | GMV / UV | 依品类而定 | 环比下滑 >20% |
| GPM (千次观看成交) | GMV / UV × 1000 | 依品类而定 | — |

## 直播间数据分析步骤

### 第一步：整场健康度诊断
1. 对比上场/上周同期数据，找出环比变化最大的 3 个指标
2. 判断问题出在哪一层（流量层 / 留存层 / 互动层 / 转化层）
3. 优先诊断最上层的问题（流量差 → 先解决流量，而非优化话术）

### 第二步：时间轴分析
如果有分时段数据（每小时/每15分钟），重点关注：
- **在线曲线**：在线人数是否稳定，峰值在什么时间点
- **GMV 曲线**：哪个时间段出单最多，对应什么产品/活动
- **流失节点**：哪个时间段在线人数骤降，分析可能原因

### 第三步：选品表现分析
逐品分析（如有SKU数据）：
- 点击率、加购率、下单率
- 找出转化漏斗断点（点击高但加购低 → 价格/详情页问题）
- 找出 GMV 贡献 top 3 的产品和原因

### 第四步：复盘结论模板

```
【本场亮点】
- [指标] 达到 [数值]，高于均值 X%，原因推测：[XXX]

【本场问题】
- [指标] 仅 [数值]，低于上场 X%，可能原因：[XXX]

【下场优化方向】
1. [优先级最高的改进项] → 具体操作：[XXX]
2. [次优先级] → 具体操作：[XXX]
```

## 常见问题诊断

| 症状 | 可能原因 | 建议方向 |
|---|---|---|
| UV 高但转化低 | 选品与受众不匹配 / 价格锚点问题 / 话术不到位 | 检查受众画像 vs 选品定位 |
| 停留时长短 | 开场节奏慢 / 主播状态差 / 内容无吸引力 | 优化开场 30 秒，增加福袋 |
| 互动率低 | 主播提问频率低 / 直播间气氛冷 | 设置互动话术，增加游戏环节 |
| 自然流占比低 | 内容质量低，系统不推 | 提升互动率，增加弹幕密度 |
| 新粉占比低 | 内容圈层固化，无破圈内容 | 设计引流款，增加短视频引流 |
