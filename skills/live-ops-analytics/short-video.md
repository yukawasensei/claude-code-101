# 短视频数据分析框架

## 核心指标体系

### 传播层
| 指标 | 健康基准（抖音参考） | 预警阈值 |
|---|---|---|
| 完播率（<1min视频） | >45% 为良好，>60% 为优秀 | <25% 需重剪开头 |
| 完播率（>1min视频） | >30% | <15% |
| 点赞率 | >3% | <1% |
| 评论率 | >0.5% | <0.1% |
| 分享率 | >1% | <0.3% |
| 关注转化率 | >1% | <0.3% |

> 以上基准仅供参考，不同品类、账号体量差异较大，建议以账号自身历史均值作为基准。

### 引流层（直播引流视频专项）
| 指标 | 说明 | 健康基准 |
|---|---|---|
| 引流率 | 点击直播间 / 播放量 | >5% |
| 引流观众留存 | 从视频进入直播间后的平均停留时长 | >3 分钟 |
| 引流转化率 | 从视频进入直播间后的下单率 | 对比直播间整体转化率 |

## 短视频分析步骤

### 第一步：内容层漏斗分析
```
播放量
  ↓ 完播率
完播人数
  ↓ 互动率（点赞+评论+分享）
互动人数
  ↓ 引流率（如果是直播引流视频）
进入直播间人数
  ↓ 转化率
成交人数
```
找出漏斗哪一层流失最严重，针对性优化。

### 第二步：内容质量诊断

**完播率低 → 检查：**
- 前 3 秒是否有强钩子（悬念/冲突/价值承诺）
- 视频节奏是否过慢
- 是否有开头广告感

**互动率低 → 检查：**
- 是否有引导互动的话术/文案
- 内容是否触发情绪（共鸣/争议/惊喜）
- 评论区是否有话题引导

**引流率低 → 检查：**
- 视频与直播间内容是否一致（承接感）
- 是否有明确的"进直播间"引导
- 视频封面/标题是否有直播引流暗示

### 第三步：账号内容矩阵分析
对比同账号多条视频时，分析：
- 哪类内容完播率最高（产品展示/教程/生活场景/对比测评）
- 哪类内容引流效率最高
- 发布时间与数据的相关性（早 8 点 vs 晚 8 点）
