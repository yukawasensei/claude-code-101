# ROI & 投流效果分析框架

## 核心投流指标

| 指标 | 计算方式 | 健康基准 | 说明 |
|---|---|---|---|
| ROI | GMV / 广告花费 | >3 为良好（依品类） | 整体投流回报 |
| ROAS | 付费 GMV / 广告花费 | 同上 | 仅计付费成交 |
| CPM | 广告花费 / 曝光量 × 1000 | 依竞争程度而定 | 千次曝光成本 |
| CPC | 广告花费 / 点击量 | — | 单次点击成本 |
| CTR | 点击量 / 曝光量 | >1.5% 为良好 | 素材吸引力指标 |
| 进直播间成本 | 广告花费 / 进播人数 | 依品类 | 拉新成本 |
| 成交成本 | 广告花费 / 成交订单数 | — | 单笔成交获客成本 |
| GPM（付费） | 付费 GMV / 付费UV × 1000 | — | 付费流量质量 |

---

## ROI 分析步骤

### 第一步：区分自然流 vs 付费流

将整场 GMV 拆分：
```
总 GMV = 自然流 GMV + 付费流 GMV
付费 ROI = 付费流 GMV / 广告花费
综合 ROI = 总 GMV / 广告花费（参考值，非真实 ROI）
```
> 重要：不要用总 GMV 计算 ROI，会虚高。

### 第二步：投放结构分析

如果有分计划/分素材数据，分析：
- 哪个计划/素材 ROI 最高 → 加量
- 哪个计划/素材 ROI 最低 → 暂停或优化
- CTR 高但 ROI 低 → 素材吸量但直播间承接差
- CTR 低但 ROI 高 → 素材精准但触达少，可扩量

### 第三步：时段投放效率分析
- 哪个时间段花费最高，ROI 是否对应最好
- 深夜投放 vs 黄金时段投放的效率对比
- 建议：将预算集中在 ROI 最高的时段

### 第四步：自然流 + 付费流协同分析
- 付费流进直播间后的停留时长 vs 自然流（判断内容承接质量）
- 付费流转化率 vs 自然流转化率（判断付费受众精准度）

## 投流诊断速查表

| 症状 | 可能原因 | 建议 |
|---|---|---|
| CPM 高，进播成本高 | 竞争激烈 / 定向太窄 | 适当放开定向 / 更换素材 |
| CTR 低 | 素材缺乏吸引力 | 更换开头，强化利益点 |
| 进播率高但转化低 | 直播间承接差 / 选品不匹配 | 优化直播间内容而非投放 |
| ROI 达标但 GMV 量小 | 预算不足 / 起量慢 | 逐步放量，观察 ROI 变化 |
| ROI 不稳定，波动大 | 竞价波动 / 直播间状态不稳定 | 稳定直播间内容，设置出价上限 |

```
【投放结构】

- 最差计划：[XXX]，ROI = X.X，建议暂停

【关键问题】
[症状] → [原因推测] → [建议动作]

【下场投放建议】

```
