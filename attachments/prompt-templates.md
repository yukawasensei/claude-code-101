## 提示词编写最佳实践

### 1. 清晰描述数据来源

✅ **好的示例**:
```
我有一份TikTok直播数据(tiktok_data.csv),包含50场直播的完整数据,
字段包括播放量、GMV、转化率等。
```

❌ **不好的示例**:
```
分析这个数据
```

### 2. 明确指定输出要求

✅ **好的示例**:
```
生成一份适合给老板看的管理层汇报,包含数据概览、关键发现、
详细分析、优化建议四个部分,每个建议都要有数据支撑。
```

❌ **不好的示例**:
```
生成一份报告
```

### 3. 提供具体的分析维度

✅ **好的示例**:
```
请分析以下维度:
1. 发布时间对播放量的影响
2. 不同内容类型的转化率对比
3. ROI和投流效率分析
```

❌ **不好的示例**:
```
分析一下数据
```

### 4. 说明设计要求

✅ **好的示例**:
```
设计要求:
- 现代简洁风格
- 蓝白配色(主色#2563eb)
- 响应式布局
- 图表支持交互
```

❌ **不好的示例**:
```
做一个好看的网页
```

### 5. 指定文件名

✅ **好的示例**:
```
文件名:
- 报告:tiktok_analysis_report.md
- 网页:tiktok_dashboard.html
```

❌ **不好的示例**:
```
保存文件
```

## 不同场景的提示词示例

### 场景1:直播复盘分析

```
我有一场直播的数据(live_data.csv),包含实时观看人数、互动数据、
成交数据等。

请使用 /live-ops-analytics skill 生成一份直播复盘报告,包含:
1. 直播概况(时长、场观、GMV)
2. 流量分析(来源、峰值时段、流失点)
3. 互动分析(弹幕、礼物、关注)
4. 转化分析(下单转化率、客单价)
5. 优化建议(话术、节奏、引导)

报告要求:
- 适合团队复盘讨论
- 重点标注问题和机会点
- 建议要具体可执行

文件名:live_review_report.md
```

### 场景2:短视频效果分析

```
我有一周的短视频数据(video_data.csv),包含10条视频的播放量、
互动数据、转化数据。

请使用 /live-ops-analytics skill 生成一份周报,包含:
1. 本周数据概览
2. 爆款视频分析(Top 3)
3. 待优化视频分析(Bottom 3)
4. 内容类型效果对比
5. 下周内容建议

报告要求:
- 简洁明了,重点突出
- 对比上周数据,标注变化
- 建议要结合实际情况

文件名:weekly_video_report.md
```

```

报告要求:

- 重点关注成本和收益
- 建议要有预期效果预测

文件名:roi_analysis_report.md
```

## 数据分析模板

### 基础统计

```
读取[文件名],分析[字段名],计算:
- 总数
- 平均值
- 中位数
- 最大值和最小值
- 标准差

生成统计摘要表格。
```

### 分组统计

```
读取[文件名],按[分组字段]分组,
统计每组的[指标字段]的[统计方法],
生成对比表格。
```

### 趋势分析

```
读取[文件名],分析[时间字段]和[指标字段]的关系,
生成趋势图,识别:
- 上升/下降趋势
- 周期性规律
- 异常值

保存为HTML格式。
```

### 相关性分析

```
读取[文件名],分析[字段A]和[字段B]的相关性,
计算相关系数,生成散点图,
说明两者的关系。
```

### 分布分析

```
读取[文件名],分析[字段名]的分布情况,
生成直方图,计算:
- 分布类型(正态/偏态)
- 集中趋势
- 离散程度

保存为HTML格式。
```

## 提示词优化技巧

### 原则1:具体明确,说明预期输出

```
❌ 分析这个文件
✅ 读取sales_data.csv,统计每个月的销售额总和,找出销售额最高的3个月,
   输出:
   1. 月度销售统计表(Markdown表格)
   2. Top 3月份柱状图(HTML格式)
   3. 趋势分析(3-5句话)
```

### 原则2:分步执行,逐步完善

```
❌ 帮我做一个完整的数据分析报告
✅
第一步:先读取数据,给我一个概览
第二步:分析销售趋势
第三步:识别异常值
第四步:生成可视化图表
第五步:撰写分析报告
```

### 原则3:提供示例和参考

```
❌ 生成一个好看的图表
✅ 生成一个柱状图,参考这个效果:
- X轴:月份(1-12月)
- Y轴:销售额(单位:万元)
- 颜色:蓝色渐变(#2563eb到#60a5fa)
- 标题:2026年月度销售趋势
- 图例位置:右上角
```

## 保存和使用提示词

### 创建提示词库文件

在你的项目文件夹中创建 `my_prompts.md`:

```markdown
# 我的提示词库

## 数据分析
[保存常用的数据分析提示词]

## 报告生成
[保存常用的报告生成提示词]

## 可视化
[保存常用的可视化提示词]
```

### 快速复用

1. 打开 `my_prompts.md`
2. 找到需要的模板
3. 复制并修改参数
4. 粘贴到 Claude Code
