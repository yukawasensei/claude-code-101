# 报告生成提示词模板

本文档提供生成专业分析报告的提示词模板。

## 数据分析报告

### 基础分析报告

```
请基于 [数据文件] 的分析结果，生成一份数据分析报告，包括：

1. **执行摘要**（200字）
   - 分析目的
   - 关键发现（3-5条）
   - 核心建议

2. **数据概览**
   - 数据来源和时间范围
   - 数据规模
   - 数据质量说明

3. **分析方法**
   - 使用的分析方法
   - 关键指标定义

4. **详细分析**
   - [维度1] 分析
   - [维度2] 分析
   - [维度3] 分析
   - 每个部分包含数据、图表和解读

5. **结论与建议**
   - 主要结论
   - 可执行的建议
   - 风险提示

6. **附录**
   - 详细数据表
   - 技术说明

请使用 Markdown 格式，适合转换为 PDF。
```

```

2. **用户分析**
   - 用户规模和增长
   - 用户活跃度
   - 用户留存情况

3. **内容/产品分析**
   - 内容表现排行
   - 爆款内容特征
   - 内容类型分布

4. **转化分析**
   - 转化漏斗
   - 转化率变化
   - 影响转化的关键因素

5. **竞品对比**（如有数据）
   - 关键指标对比
   - 优劣势分析

6. **下月策略**
   - 优化方向
   - 具体行动计划
   - 预期目标

格式要求：专业、简洁，配图表。
```

## 报告撰写技巧

### 1. 结构清晰

```
好的报告结构：
- 金字塔原理：结论先行
- 逻辑清晰：总-分-总
- 层次分明：使用标题和编号
- 重点突出：使用加粗、颜色
```

### 2. 数据可视化

```
数据呈现原则：
- 关键数据用图表展示
- 图表简洁清晰
- 配文字说明
- 突出关键信息
```

### 3. 语言表达

```
专业表达：
- 客观准确：基于数据，避免主观臆断
- 简洁明了：避免冗长和术语堆砌
- 逻辑严密：因果关系清晰
- 结论明确：不模棱两可
```

### 4. 建议可执行

```
好的建议应该：
- 具体：明确做什么
- 可行：考虑资源和约束
- 有优先级：分清轻重缓急
- 有预期：说明预期效果
```

```
- 平均转化率 X%，环比增长 Y%
- 教程类视频转化效果最好
- 建议增加教程类内容占比

### 数据概览
- 数据来源：TikTok 后台
- 时间范围：2026-01-01 至 2026-01-31
- 视频数量：50条
- 总播放量：XXX万

### 内容表现分析

#### 1. 视频类型分析
[插入柱状图：各类型平均转化量]

- 教程类：平均转化 XX 次，占比 XX%
- 开箱类：平均转化 XX 次，占比 XX%
- ...

**洞察**：教程类视频转化率最高，但数量较少。

#### 2. 发布时间分析
[插入热力图：发布时间 vs 转化效果]

- 最佳发布时间：18:00-20:00
- 周末转化效果优于工作日

**洞察**：晚间发布效果最好，建议调整发布计划。

#### 3. 内容特征分析
[插入散点图：播放量 vs 转化量]

高转化视频的共同特征：
1. 视频时长 30-60 秒
2. 前 3 秒有吸引力
3. 有明确的行动号召

### 结论与建议

#### 主要结论
1. 教程类内容转化效果最好
2. 发布时间对转化有显著影响
3. 视频时长和开头很关键

#### 下月策略

   - 增加教程类视频占比至 40%

   - 控制视频时长在 30-60 秒

2. **发布策略**
   - 主要在 18:00-20:00 发布
   - 周末增加发布频次

3. **预期目标**
   - 平均转化率提升 20%
   - 总转化量达到 XXX

### 附录
[详细数据表]

---
报告生成时间：2026-02-01
分析师：[姓名]
```

## 常见问题

### Q: 报告应该多长？
A: 根据受众和目的：
- 高管报告：1-2页，重点突出
- 详细分析报告：5-10页，数据完整
- 专题研究报告：10-20页，深度分析

### Q: 如何让报告更专业？
A:
1. 使用专业术语（但要解释）
2. 数据支撑每个结论
3. 图表清晰美观
4. 格式统一规范

### Q: 如何提高报告的可读性？
A:
1. 使用标题和小标题
2. 重点内容加粗或高亮
3. 适当使用列表
4. 配图表说明
