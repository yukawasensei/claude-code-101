# 可视化提示词模板

本文档提供数据可视化的提示词模板，帮助你快速生成专业的图表。

## 基础图表

### 柱状图

```
请生成柱状图展示 [数据]：
1. X轴：[类别字段]
2. Y轴：[数值字段]
3. 标题：[图表标题]
4. 按 [排序方式] 排序
5. 使用合适的颜色方案
```

### 折线图

```
请生成折线图展示 [指标] 的趋势：
1. X轴：[时间字段]
2. Y轴：[数值字段]
3. 如果有多个系列，请用不同颜色区分
4. 标注关键时间点
5. 添加趋势线
```

### 饼图

```
请生成饼图展示 [类别] 的占比：
1. 显示百分比
2. 按占比从大到小排序
3. 使用清晰的颜色区分
4. 如果类别太多，将小类别合并为"其他"
```

### 散点图

```
请生成散点图分析 [变量1] 和 [变量2] 的关系：
1. X轴：[变量1]
2. Y轴：[变量2]
3. 如果有分类，用不同颜色标记
4. 添加趋势线
5. 标注异常点
```

## 高级图表

### 热力图

```
请生成热力图展示 [数据] 的相关性：
1. 计算相关系数矩阵
2. 使用颜色深浅表示相关性强度
3. 在每个格子中显示数值
4. 使用合适的颜色方案（红蓝渐变）
```

### 箱线图

```
请生成箱线图对比不同 [类别] 的 [指标] 分布：
1. X轴：[类别]
2. Y轴：[指标]
3. 显示中位数、四分位数、异常值
4. 使用不同颜色区分类别
```

### 堆叠柱状图

```
请生成堆叠柱状图展示 [数据]：
1. X轴：[主类别]
2. Y轴：[数值]
3. 堆叠维度：[子类别]
4. 显示每个部分的占比
5. 添加图例
```

## 组合图表

### 双轴图

```
请生成双轴图同时展示 [指标1] 和 [指标2]：
1. 左Y轴：[指标1]（柱状图）
2. 右Y轴：[指标2]（折线图）
3. X轴：[时间/类别]
4. 使用不同颜色区分两个指标
```

### 仪表盘

```
请创建一个数据仪表盘，包含：
1. [图表1类型]：展示 [数据1]
2. [图表2类型]：展示 [数据2]
3. [图表3类型]：展示 [数据3]
4. [图表4类型]：展示 [数据4]
5. 使用统一的配色方案
6. 保存为 HTML 文件，可交互
```

## 图表优化

### 美化图表

```
请优化现有图表：
1. 调整图表大小为 [宽度] x [高度]
2. 使用 [配色方案]
3. 添加网格线
4. 优化标签字体大小
5. 添加数据标签
6. 调整图例位置
```

### 交互式图表

```
请生成交互式图表：
1. 使用 Plotly 库
2. 支持鼠标悬停显示详细信息
3. 支持缩放和平移
4. 支持图例点击切换显示
5. 保存为 HTML 文件
```

## 使用示例

### 示例 1: TikTok 数据可视化

```
请为 TikTok 数据生成以下可视化：

1. 柱状图：各视频类型的平均转化量
   - X轴：视频类型
   - Y轴：平均转化量
   - 按转化量降序排列

2. 散点图：播放量 vs 转化量
   - X轴：播放量
   - Y轴：转化量
   - 用不同颜色标记视频类型
   - 添加趋势线

3. 折线图：每日转化量趋势
   - X轴：日期
   - Y轴：转化量
   - 标注转化量最高的日期

4. 热力图：发布时间 vs 转化效果
   - X轴：发布小时
   - Y轴：星期几
   - 颜色深浅表示平均转化量

请将所有图表保存到 visualizations/ 目录。
```

### 示例 2: 项目进度可视化

```
请为项目进度数据生成仪表盘：

1. 折线图：8周的完成率趋势
   - 显示计划完成率和实际完成率
   - 标注关键里程碑

2. 堆叠柱状图：每周的 Bug 情况
   - 显示发现的 Bug 和修复的 Bug
   - 用不同颜色区分

3. 折线图：团队速度变化
   - 显示每周的团队速度
   - 添加平均线

4. 柱状图：测试覆盖率提升
   - 显示每周的测试覆盖率
   - 标注目标线（80%）

将所有图表组合成一个 HTML 仪表盘。
```

## 图表类型选择指南

| 数据类型 | 推荐图表 | 用途 |
|---------|---------|------|
| 分类对比 | 柱状图 | 对比不同类别的数值 |
| 时间趋势 | 折线图 | 展示数据随时间变化 |
| 占比分布 | 饼图 | 展示各部分占整体的比例 |
| 相关关系 | 散点图 | 分析两个变量的关系 |
| 分布情况 | 直方图/箱线图 | 展示数据的分布特征 |
| 多维对比 | 热力图 | 展示多个维度的数据 |
| 层次结构 | 树状图 | 展示层级关系 |
| 地理分布 | 地图 | 展示地理位置数据 |

## 常见问题

### Q: 图表太拥挤怎么办？
A: 可以调整图表大小、减少数据点、或者分成多个图表展示。

### Q: 如何选择合适的图表类型？
A: 根据数据类型和分析目的选择，参考上面的"图表类型选择指南"。

### Q: 图表无法显示怎么办？
A: 确保安装了必要的库（matplotlib, seaborn, plotly），Claude Code 会自动安装。

### Q: 如何导出高清图片？
A: 在生成图表时指定 `dpi=300` 参数，或者保存为 SVG 格式。
