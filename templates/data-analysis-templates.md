# 数据分析提示词模板

本文档提供常用的数据分析提示词模板，帮助你快速开始数据分析任务。

## 基础数据探索

### 模板 1: 数据概览

```
请帮我分析 [文件名] 文件：
1. 数据概览（行数、列数、字段类型）
2. 检查是否有缺失值和异常值
3. 显示前 10 行数据
4. 给出每个字段的基本统计信息
```

### 模板 2: 数据质量检查

```
请检查 [文件名] 的数据质量：
1. 缺失值统计（每列的缺失比例）
2. 重复数据检查
3. 异常值识别（使用箱线图方法）
4. 数据类型是否正确
5. 给出数据清洗建议
```

## 描述性统计

### 模板 3: 基础统计分析

```
请对 [文件名] 进行统计分析：
1. 数值型字段的均值、中位数、标准差、最大最小值
2. 分类型字段的频数分布
3. 识别数据的分布特征（正态分布、偏态等）
```

### 模板 4: 分组统计

```
请按 [分组字段] 对数据进行分组统计：
1. 每组的数量
2. 每组的 [目标字段] 的均值、中位数
3. 组间差异分析
4. 找出表现最好和最差的组
```

## 相关性分析

### 模板 5: 相关性矩阵

```
请分析 [文件名] 中各数值字段之间的相关性：
1. 计算相关系数矩阵
2. 生成相关性热力图
3. 找出相关性最强的字段对（正相关和负相关）
4. 解释相关性的业务含义
```

### 模板 6: 目标变量相关性

```
请分析哪些因素与 [目标变量] 相关：
1. 计算所有字段与目标变量的相关系数
2. 按相关性强度排序
3. 生成散点图展示关键相关关系
4. 给出影响目标变量的前 5 个因素
```

## 趋势分析

### 模板 7: 时间序列分析

```
请分析 [指标名称] 随时间的变化趋势：
1. 绘制时间序列折线图
2. 计算增长率（环比、同比）
3. 识别趋势（上升、下降、平稳）
4. 找出异常时间点
5. 预测未来趋势
```

### 模板 8: 周期性分析

```
请分析 [指标名称] 的周期性规律：
1. 按 [时间维度：小时/星期/月份] 分组统计
2. 绘制周期性图表
3. 找出高峰和低谷时段
4. 给出业务建议
```

## 对比分析

### 模板 9: A/B 对比

```
请对比 [组A] 和 [组B] 的差异：
1. 样本量对比
2. 关键指标对比（均值、中位数）
3. 分布对比（箱线图或直方图）
4. 统计显著性检验
5. 结论和建议
```

### 模板 10: 多维度对比

```
请从多个维度对比不同 [分类] 的表现：
1. [维度1] 的对比
2. [维度2] 的对比
3. [维度3] 的对比
4. 综合评分
5. 找出最优和最差的类别
```

## 异常检测

### 模板 11: 异常值识别

```
请识别 [文件名] 中的异常数据：
1. 使用统计方法（3σ原则、IQR方法）识别异常值
2. 列出所有异常记录
3. 分析异常的可能原因
4. 建议如何处理这些异常值
```

### 模板 12: 异常模式分析

```
请分析 [指标名称] 的异常模式：
1. 识别异常时间点或异常样本
2. 分析异常发生的规律
3. 找出异常的共同特征
4. 给出预警建议
```

## 细分分析

### 模板 13: 用户细分

```
请对用户进行细分分析：
1. 基于 [特征1, 特征2, 特征3] 进行聚类
2. 描述每个细分群体的特征
3. 计算每个群体的规模和价值
4. 给出针对性的运营建议
```

### 模板 14: RFM 分析

```
请进行 RFM 分析：
1. 计算每个用户的 Recency（最近购买时间）、Frequency（购买频次）、Monetary（购买金额）
2. 对 RFM 进行评分和分组
3. 识别高价值用户、流失风险用户等
4. 给出用户运营策略
```

## 漏斗分析

### 模板 15: 转化漏斗

```
请分析 [业务流程] 的转化漏斗：
1. 计算每个环节的转化率
2. 识别流失最严重的环节
3. 对比不同 [维度] 的转化差异
4. 给出优化建议
```

## 使用技巧

### 1. 明确目标
在使用模板前，先明确分析目标：
- 你想回答什么问题？
- 需要什么样的输出？
- 分析结果将如何使用？

### 2. 调整模板
根据实际情况调整模板：
- 替换 [占位符] 为实际的文件名、字段名
- 增加或删除分析步骤
- 调整分析的深度和广度

### 3. 迭代优化
根据 Claude 的输出继续提问：
- "请详细解释 [某个发现]"
- "请针对 [某个问题] 进行深入分析"
- "请给出更具体的建议"

### 4. 组合使用
将多个模板组合使用：
1. 先用"数据概览"了解数据
2. 再用"相关性分析"找关键因素
3. 最后用"对比分析"验证假设

## 示例：完整分析流程

```
# 第1步：数据探索
请帮我分析 sales_data.csv 文件：
1. 数据概览（行数、列数、字段类型）
2. 检查是否有缺失值和异常值
3. 显示前 10 行数据

# 第2步：描述性统计
请对销售额、订单量等关键指标进行统计分析：
1. 均值、中位数、标准差
2. 最大最小值
3. 分布特征

# 第3步：趋势分析
请分析销售额随时间的变化趋势：
1. 绘制月度销售额折线图
2. 计算环比增长率
3. 识别增长或下降趋势

# 第4步：细分分析
请按产品类别分析销售表现：
1. 每个类别的销售额和订单量
2. 类别间的对比
3. 找出表现最好的类别

# 第5步：生成报告
请基于以上分析生成销售分析报告，包括：
1. 数据概览
2. 关键发现
3. 趋势分析
4. 建议和行动计划
```

## 常见问题

### Q: 模板太长，Claude 会执行所有步骤吗？
A: 会的。Claude 会按步骤执行，如果某个步骤需要很长时间，可以分开提问。

### Q: 如何让分析更深入？
A: 在模板基础上增加"为什么"的问题，例如"请分析造成这种现象的可能原因"。

### Q: 分析结果不符合预期怎么办？
A: 提供更多上下文信息，或者明确指出你期望的分析方向。
